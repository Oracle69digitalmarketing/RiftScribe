---

**Goal:** Build a feature-complete, polished frontend prototype to define the user experience.

-   [x] **Task 0.1:** Design and build the main UI/UX flow.
-   [x] **Task 0.2:** Implement AI persona selection.
-   [x] **Task 0.3:** Implement real-time AI saga and image generation.
